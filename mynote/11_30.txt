Java
변수
-프로그램 작업을 처리하기 위한 값을 저장할 수 있는 메모리 공간
-타입을 지정할 수 있고 메모리 공간에 이름을 붙여 관리한다.
ex) 	int(타입)	i(변수명)	= 10(값);

2)변수 타입
-boolean (1byte) : true/false로 나뉜다.
-char (2byte) : 문자 한 글자
-short (2byte) / int (4byte) / long (8byte) : 정수
-float(4byte) / double(8byte): 실수

3) 선언 방법
자료형 변수명: 선언과 초기화 분리 ex) int num1; num1=10;
자료형 변수명 = 값: 선언과 동시에 초기화 ex) int num1 = 20;
자료형 변수명1, 2: 동일한 자료형 다수개 선언 ex) int num3, num4;
자료형 변수명1=값1, 변수명2=값2: 동일한 자료형 다수개 선언 및 초기화 ex)int num5, num6 = 50, num7 = 70;

4) 기본 / 참조 데이터 타입
-기본 데이터 타입: 기존의 데이터 타입들
-참조 데이터 타입: String > 객체 자료형으로 크기가 크고 복잡하여 저장된 주소를 가리켜 메모리에 할당한다.

5)형변환
-묵시적 형변환: 작은 크기의 데이터를 큰 메모리 공간의 데이터 타입으로 옮긴다.
-명시적 형변환: 묵시적 형변환과 반대이며 데이터 손실이 있을 수 있다.
