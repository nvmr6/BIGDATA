4) 반복문
:조건이 참이기 전까지 반복적으로 수행

-for(int num = 0;num<10;num++){
			명령문
}
: 설정한 횟수만큼 반복하며 for문 안에 for문이나 다른 조건문을 넣어 활용한다.
-break / continue: 조건에 따라 정지하거나 생략

-while(조건){
	명령문
}
:조건식이 참이기 전까지 반복 수행
-do-while: 최초 한번은 무조건 실행하고 두번째부터는 조건이 참일시 수행

5)배열
:동일 자료형의 집합, 다수개의 데이터 사용 가능
-{}을 사용하며 최초에 배열의 크기를 설정한다.
-참조변수를 이용하여 배열을 관리한다.
ex) int[] 배열 변수 이름 = new int[5]; > 배열 변수 선언과 메모리 할당
ex)int[] score = {10,20,30,40,50};
int[] s = new int[score.length]; 
for(int i=0;i<score.length;i++) {
	s[i] = score[i];
} > 배열 복사: 배열을 같은 메모리 크기로만 만들고 값끼리 복사
